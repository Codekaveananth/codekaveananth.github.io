🏥 Multi-Hospital Performance Dashboard (Power BI)

📌 Project Overview

The Multi-Hospital Performance Dashboard is an end-to-end data analytics project built using Power BI to analyze and visualize hospital operations, patient trends, staffing efficiency, and workload performance.

This dashboard enables stakeholders to make data-driven decisions by providing insights across multiple hospitals through interactive and visually intuitive reports.

---

🎯 Objectives

Analyze patient data across hospitals
Compare hospital performance and efficiency
Evaluate staffing distribution and resource utilization
Monitor workload trends and emergency cases
Provide interactive filtering for dynamic insights

📊 Dashboard Pages

🔹 1. Executive Overview

Total Patients, Total Billing, Avg Billing, Avg Stay
Patient distribution by hospital
Hospital rating & ownership breakdown
Global slicers (State, City, Ownership)

🔹 2. Patient Analysis

Emergency % and Abnormal Results %
Patient distribution by admission type & medical condition
Billing analysis by condition
Monthly patient trend analysis

🔹 3. Hospital Comparison

Revenue comparison across hospitals
Avg stay efficiency analysis
Rating vs billing correlation (scatter plot)
Market share distribution
Detailed hospital performance matrix

🔹 4. Staff & Operations

Total staff and staffing efficiency metrics
Staff distribution by role (Doctor, Nurse, etc.)
Department-wise staffing heatmap
Staff-to-patient ratio comparison

🔹 5. Workload Trends

Total treated patients and ICU rate
Emergency case trends
Weekly workload analysis
Week-over-week growth insights


🧮 Key DAX Measures

Total Patients
Total Billing
Avg Billing per Patient

Emergency %

WoW Growth %

🏗️ Data Model

Star schema design
Central dimension: Hospital Info
Fact tables: Patient, Staff, Workload
Dedicated Date Table for time intelligence
